> [!info] Como Usar Este Plano
> Defina seus treinos usando cabeçalhos de nível 3 (`###`). Para cada exercício, use o formato exato:
> `- Nome do Exercício @ SériesxReps (Descanso em segundos + 's')`
> 
> **Exemplos:**
> - Supino Reto com Barra @ 3x5 (120s)
> - Flexão @ 3xAMRAP (90s)  *(Use "AMRAP" para "As Many Reps As Possible")*
> - Corrida Leve @ 1x30min (0s) *(Use "min" para atividades contínuas)*
