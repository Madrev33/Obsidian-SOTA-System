# 🧠 Central de Processamento de Ideias

> [!multi-column]
>
>> [!info] Tasks Fazer Depois
>>```dataviewjs
>>const tasks = dv.pages().file.tasks
>>    .where(t => !t.completed && t.text.includes("#fazer_depois"))
>>    // Ordena de forma segura
>>    .sort(t => {
>>        // Tenta pegar a data do arquivo (file.day)
>>        // Se não existir (arquivo sem data no nome), usa a data de criação (file.ctime)
>>        const file = dv.page(t.path)?.file;
>>        return file?.day || file?.ctime;
>>    }, 'asc');
>>if (tasks.length > 0) {
>>    dv.header(4, `📥 Capturas Pendentes (${tasks.length})`);
>>    dv.taskList(tasks, false);
>>} else {
>>    dv.paragraph("✅ Nenhuma captura pendente para processar.");
>>}
>>```
